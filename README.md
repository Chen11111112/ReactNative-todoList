🛠 開始前的準備事項 (Prerequisites)
在執行 npx expo start 之前，請務必完成以下配置：

2. 基本環境
Node.js: v18.x 或更高。

Expo Go App: 請在實體手機下載 Expo Go。

3. 安裝專案套件
下載專案後，請先安裝所有依賴：
npm install
🚀 啟動步驟
請按照以下指令啟動專案：

進入專案資料夾
cd your-project-name

以 Tunnel 模式啟動
npx expo start --tunnel

掃描 QR Code

等待終端機產出 URL（會顯示為 exp://u.expo.dev/...）。

使用 iPhone 相機或 Expo Go 掃描 QR Code。


你好
